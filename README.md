# lfa_controle_combos_afd
Este trabalho visa aplicar os conceitos de "Linguagens Formais e Autômatos" na
prática, utilizando autômatos finitos. Segundo Sipser (2005), autômatos finitos são
modelos matemáticos que representam sistemas capazes de transitar entre estados em
resposta a entradas específicas. Compostos por um conjunto de estados, um alfabeto de
entrada, uma função de transição, um estado inicial e um conjunto de estados finais, os
autômatos finitos oferecem uma estrutura formal para entender e modelar processos de
mudança de estado.
Os Autômatos Finitos Determinísticos, ou AFDs, possuem uma base teórica rica
em termos de desenvolvimento e uso matemático. Desta forma, existem diversas
aplicações para o uso de autômatos finitos. As aplicações dos AFDs incluem análise de
protocolos, análise de texto, comportamento de personagens em vídeo games, análise de
segurança, unidades de controle de CPU, processamento de linguagem natural e
reconhecimento de fala. Além disso, muitos dispositivos mecânicos simples (e não tão
simples) são frequentemente projetados e implementados utilizando AFDs, como
elevadores, máquinas de venda automática e semáforos sensíveis ao tráfego
(GRIBKOFF, 2013).
No contexto dos videogames, os AFDs podem ser usados para controlar o
comportamento de personagens, permitindo que eles realizem uma série de ações, como
golpes e movimentos especiais, de maneira determinística e previsível. Além disso, um
dos principais desafios é garantir que o AFD seja suficientemente complexo para
permitir uma variedade de movimentos e golpes, mas também suficientemente simples
para ser eficiente e fácil de implementar.
Além disso, o AFD deve ser projetado de tal forma que permita uma
jogabilidade equilibrada, garantindo que nenhum personagem ou estratégia seja
excessivamente dominante (SMITH, 2010). Interessante ressaltar que Jogos populares
como “Street Fighter” e “Mortal Kombat” utilizam AFDs para controlar o
comportamento de seus personagens, e esses jogos são frequentemente citados como
exemplos de excelência em design de jogos de luta (DIXIT, 2015).
No âmbito deste trabalho, propõe-se a construção de uma aplicação em Python
que simula o controle de personagens em um jogo de luta. Os movimentos do
personagem serão modelados com um autômato finito, capaz de responder às entradas
do jogador e transitar entre diferentes estados de acordo com as ações.
A utilização de autômatos finitos neste contexto permite simular o
comportamento de um personagem, proporcionando uma experiência de jogo mais
dinâmica e desafiadora. A interação entre os autômatos dos personagens e as entradasdo jogador cria um ambiente virtual onde estratégia, reflexos e habilidades são postos à
prova.
